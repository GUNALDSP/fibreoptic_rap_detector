Installation instructions for Proteus Professional v8.16 SP3:

1. Disconnect from the Internet
2. Extract all files from the Fix.zip archive to a known location.
3. Run the installation proteus8.16.SP3.exe
4. During the installation select and install the license file Licence2.lxk
5. Continue installation and after finish it DO NOT START THE PROGRAM!
6. Apply Patch-Proteus-8.16-SP3-36097.0.exe patch.
7. Now you can turn on the Internet and start the Proteus program.
(Please note that this patch does not work in WinXP, Win7)
